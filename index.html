Array

An array is a special kind of object. It is an ordered collection.

There are two syntaxes for creating an empty array: [] and new Array(). But new Array(3) creates array with length 3.

Get last elements - arr.at(-1) or arr[arr.length - 1]

Methods push/ pop, shift/ unshift. Methods push/pop run fast, while shift/unshift are slow, because elements need to be renumbered.

Loops - the oldest way for loop, for..of and for..in. the last one is actually a bad idea.

Array length - the greatest numeric index + 1. It can manualy changed. If we decrease length, the array is truncated

Conversation - arrays implement only toString conversion

Comparison - The == operator doesn't do item-by-item comparison. That's simple: don't use the == operator. Instead, compare them item-by-item in a loop or using iteration methods

Array methods

To add/remove elements: push(...items), pop(), shift(), unshift(...items), splice(pos, deleteCount, ...items), slice(start, end), concat(...items)

To search among elements: indexOf/lastIndexOf(item, pos), includes(value), find/filter(func), findIndex()

To iterate over elements: forEach(func)

To transform the array: map(func), sort(func), reverse(), split/join, reduce/reduceRight(func, initial)
